# CODSOFT Internship

## Overview

This repository contains the machine learning projects completed as part of the **CodSoft Machine Learning Internship**. These projects demonstrate the practical application of machine learning techniques for solving real-world problems through data preprocessing, exploratory data analysis, model training, evaluation, and prediction.

Throughout this internship, I worked with different types of datasets and implemented both classification and regression algorithms using Python and popular machine learning libraries. The projects helped me gain hands-on experience in building end-to-end machine learning solutions and understanding how data-driven approaches can be used to make intelligent predictions.

---

## Projects

### Task 1: Titanic Survival Prediction

#### Objective

To develop a machine learning model that predicts whether a passenger survived the Titanic disaster based on passenger information such as age, gender, passenger class, and fare.

#### Key Activities

* Data Cleaning and Preprocessing
* Exploratory Data Analysis (EDA)
* Feature Selection and Encoding
* Model Training and Testing
* Performance Evaluation using Accuracy, Precision, Recall, and F1-Score

#### Outcome

Successfully built a classification model capable of predicting passenger survival and identifying key factors that influenced survival outcomes.

---

### Task 2: Movie Rating Prediction

#### Objective

To build a machine learning model that predicts movie ratings based on features such as genre, director, and actors.

#### Key Activities

* Data Cleaning and Feature Engineering
* Handling Missing Values
* One-Hot Encoding of Categorical Features
* Regression Model Training
* Performance Evaluation using MAE, RMSE, and R² Score

#### Outcome

Successfully developed a movie rating prediction system capable of estimating ratings based on movie characteristics and historical data.

---

### Task 3: Credit Card Fraud Detection

#### Objective

To develop a machine learning system that can identify fraudulent credit card transactions and distinguish them from genuine transactions.

#### Key Activities

* Data Preprocessing and Normalization
* Handling Class Imbalance using Undersampling
* Logistic Regression and Random Forest Implementation
* Model Evaluation using Precision, Recall, F1-Score, and Confusion Matrix

#### Outcome

Successfully created a fraud detection model that accurately classified fraudulent and genuine transactions while demonstrating the importance of handling imbalanced datasets.

The Credit Card Fraud Detection model was developed using Logistic Regression and Random Forest algorithms. The dataset was preprocessed and normalized, and class imbalance was handled using undersampling. Performance was evaluated using Precision, Recall, and F1-score. Both models achieved an F1-score of 0.967, indicating excellent performance in detecting fraudulent transactions. Since the performance was identical, either model can be selected for deployment.
